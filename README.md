# 🧑‍💻 Étienne — Coding & Ethical Hacking Assistant

A French-personality AI chatbot for coding and ethical hacking, built as a Progressive Web App (PWA).

## 🌐 Live App
👉 https://sites.super.myninja.ai/166158da-40c5-465e-b14b-b59bd9858374/4e69dc40/index.html

## 📱 Install on iPhone (PWA)
1. Open Safari on your iPhone
2. Visit the live URL above
3. Tap Share → Add to Home Screen
4. Étienne appears as a native app!

## 🔄 Auto-Deploy
Every push to `main` branch auto-deploys via GitHub Actions to GitHub Pages:
👉 https://jhwood420.github.io/hacker-bot-etienne/

## 🚀 Local Development
```bash
git clone git@github.com:jhwood420/hacker-bot-etienne.git
cd hacker-bot-etienne
python3 -m http.server 8181
```
Then open http://localhost:8181
